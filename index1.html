What is CSS ?

  1. CSS Stands for cascading style sheets.
  2. CSS describe how HTML element are to be displayed on scren, paper, or in other media.
  3. CSS is not neccesary CSS saves a lot of work.It can control the layout of multiple web pages all at once
  4. External stylesheet are stored in css file.

CSS can be added to HTML document in 3 ways:

  • inline - by using the style attribute inside HTML element1
  • internal - by using a style element in the head section
  • External - by using a link element to a link to an external CSS file.
  • The most comman way to add css, is to keep the style in external CSS file , HOWEVER in this tutorial will be use inline and internal styles, because this is easier to demostrate , and easier for you to try it yourself.

When comparing CSS class vs ID, the diffrence is that CSS class applies a style to multiple elemente. ID on the other hand, on the other hand, applies a style to one unique element .ID is also special in that you can use a special URL to link directly to an element and it's used by JavaScript .


In CSS, selectors are used to target a specific element or range of element on a web page.Once an element has been targeted ,a style or set of style can be applioed to the element .


There is a wide range of selector available. Two of the most commonly used are Class and ID . Both are used to target elements to which a style should be applied .
